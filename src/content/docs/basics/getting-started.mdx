Clack is a modern, flexible and powerful CLI library that helps you build beautiful command-line interfaces with ease.
It provides a set of high-level components and low-level primitives that make it simple to create interactive command-line applications.

## Installation

You can install Clack using npm, yarn, or pnpm.
